# 第12周实验：偏微分方程 (PDE) —— 椭圆与双曲型方程

本周实验聚焦于计算物理中两类经典的偏微分方程：**椭圆型（静电势问题）**与**双曲型（波动方程）**。我们将通过数值迭代法求解稳态场，并探讨波动传播中的稳定性条件。

## 2. 任务详细说明

### Task 1: Jacobi vs Gauss-Seidel (椭圆型)
**背景**：考虑课件中的矩形静电场问题：上边界电势 100 V，其余三边 0 V，内部无电荷（满足拉普拉斯方程 $\nabla^2 V = 0$）。
- **物理理解**：写出内部点的五点菱形离散格式，并用物理语言解释“内部点等于四个邻点平均值”的含义。
- **算法实现**：分别实现 Jacobi（同步更新，使用旧值）和 Gauss-Seidel（异步更新，使用最新值）迭代。
- **性能对比**：在网格 $N=40$、容差 $10^{-3}$ 下，对比 Jacobi 与“朴素 for 循环版” Gauss-Seidel 的迭代次数与实际运行时间。
- **深度思考**：为什么从数学上收敛更快的 Gauss-Seidel，在 Python 原生循环中不一定比向量化的 Jacobi 快？请从“数据依赖”与“向量化”的角度进行分析。

### Task 2: 红黑排序向量化 SOR (椭圆型)
**背景**：Gauss-Seidel 虽好，但其异步更新特征阻碍了 NumPy 的向量化。
- **数学准备**：写出 SOR 格式的更新公式，并说明松弛因子 $\omega$ 取值（ $\omega=1$ vs $1<\omega<2$）的物理意义。
- **红黑排序**：实现红黑排序（Red-Black Ordering），将网格点分为两类，从而在保持 Gauss-Seidel 逻辑的同时实现全向量化计算。
- **参数扫描**：在 $N=80$、容差 $10^{-4}$ 下，测试 $\omega=1.0, 1.4, 1.8$ 的表现，寻找当前问题下的“最优松弛因子”。
- **发散分析**：思考并解释为什么当 $\omega \ge 2$ 时算法会发生剧烈发散。

### Task 3: CFL 条件与“数值灾难” (双曲型)
**背景**：一维波动方程 $\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2}$ 的稳定性受库朗数 $c = a\Delta t / \Delta x$ 严格限制。
- **数值实验**：编写程序对比 $c=0.5, 0.99, 1.01$ 三种情况下的波动演化。
- **灾难现场**：复现 $c=1.01$ 时的数值爆炸（出现锯齿、`NaN` 或振幅失控）。
- **冷启动细节**：说明你如何处理 $t = \Delta t$ 的第一步启动。解释为什么简单的 $u(x, \Delta t) = u(x, 0)$ 会导致精度下降甚至产生非物理波形。
